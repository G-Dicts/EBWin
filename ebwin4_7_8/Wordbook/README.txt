2) NGSL,BSL,TSL ( http://www.newgeneralservicelist.org/ )
	NEW_GENERAL_SERVICE_LIST_1.01.txt
	THE_BUSINESS_SERVICE_LIST_1.01.txt
	THE_TOEIC_SERVICE_LIST_1.1.txt

〇使用方法
メニュー→編集(E)→単語帳のインポート(W)...
単語帳テキストファイルを選択する。
もしくは、単語帳ペインにエクスプローラからドラッグ＆ドロップする

単語帳を表示して単語を選択すると、現在選択されている辞書から検索する


〇単語帳の形式

エンコード：UTF-8
一行に単語一件のテキストファイル。

検索語<CR>

　または

検索語<TAB>レベル<CR>

レベルは1..３の数字。レベルを指定した場合、単語帳に☆,☆☆,☆☆☆を表示する


【例】
able
about
above
accept
according
account
accuse
...
