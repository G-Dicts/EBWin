EBWin 4 - EPWING/EB viewer for Windows 

■概要

- EBWin4は、Windows 用のEPWING/電子ブック(EB) ビューアです。
- 従来のEBWinの後継ソフトであり、最近のEBPocket(iOS,Android),EBMac(Mac OS X)の改良を反映しています。

■ インストール方法

- setup.exe を実行し、画面の指示に従ってインストールを進めてください。
- Windows8.1以降では、インターネットからダウンロードされたソフトウェアを実行しようとすると、
Windows SmartScreenによって「WindowsによってPCが保護されました。
Windows SmartScreenは認識されないアプリの起動を停止しました。
このアプリを実行すると、PCに問題が起こる可能性があります。」と表示され、インストールが中断します。
この場合、[詳細情報]を選択し、[実行]を押すとインストールを続行できます。

■ 使用方法

初回起動時、サンプル辞書としてEDICT2が登録されます。

1) 辞書の登録

- [ファイル] → [辞書の追加] を選択します。
- ファイルダイアログが開くので、EPWING辞書のCATALOGSを選択します。 
　LogoVista電子辞典の場合は 辞書名.idx , PDIC辞書は *.DIC, StarDict辞書は *.ifo を選択します。 
- [ファイル(F)] → [辞書の編集(E)]を選択すると、辞書の編集(追加・削除・順序の変更)ができます。
- [ファイル(F)] → [辞書の再検索(R)]では、指定したディレクトリ以下のEPWING辞書を一括して登録できます。 


2) 検索
- 検索する辞書を、ツールバーか、[辞書(D)]メニューから選択します。[すべて]を選択すると、グループ内の全ての辞書を対照に串刺し検索を行います。 
- 検索語欄にキーワードを入力し、Enterまたは検索ボタンを押します。 
- 検索結果の一覧が表示されるので、項目を選択すると本文が表示されます。 
- 本文中の参照リンク（青文字）を押すと、関連する項目を表示できます。 
- 検索方法を指定する場合は、ツールバーか、[検索(S)]メニューから選択してください。 

■辞書の圧縮

- 同梱のEBShrinkを使用すると、EPWING辞書をezip形式に圧縮することができます。
